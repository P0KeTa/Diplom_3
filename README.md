# Диплом UI и автотесты на Selenium

## 📋 Описание
Проект для автоматизации тестирования веб-приложение Stellar Burgers с использованием Selenium WebDriver и JUnit.

## 🔧 Технологии
- Java 11
- Maven
- Selenium 4.33.0
- JUnit 4
- ChromeDriver
- YandexDriver

## 🧪 Что покрыто автотестами
- Регистрация
- Вход
- Раздел «Конструктор»

## 🚀 Как запустить
1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/P0KeTa/Diplom_3.git
2. Перейти в директорию проекта и запустить тесты:
   ```bash
   mvn clean test

## 📂 Структура проекта
- pages/ — Page Object модели
- tests/ — JUnit тесты
